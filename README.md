# Web Development Project 5 - _GameCharts_

Submitted by: **Ethan Jacinto**

This web app: **provides a list of highly rated games for the user to enjoy**

Time spent: **18** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **The list displays a list of data fetched using an API call**
- [x] **Data uses the useEffect React hook and async/await syntax**
- [x] **The app dashboard includes at least three summary statistics about the data such as**
  - [x] _Total number of games_
  - [x] _Average rating of games_
  - [x] _The Game With the Most Owned Copies (Maximum)_
- [x] **A search bar allows the user to search for an item in the fetched data**
- [x] **Multiple different filters (2+) allow the user to filter items in the database by specified categories**

The following **optional** features are implemented:

- [ ] Multiple filters can be applied simultaneously
- [x] Filters use different input types such as a text input, a selection, or a slider
- [ ] The user can enter specific bounds for filter values

The following **additional** features are implemented:

- [x] Utilization of Tailwind for a better UI design
