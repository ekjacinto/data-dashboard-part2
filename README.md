# Web Development Project 6 - _GameCharts 2.0_

Submitted by: **Ethan Jacinto**

This web app: **provides a list of highly rated games, including data visualizations and expanded page views, for the user to enjoy**

Time spent: **18** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **The app includes at least one unique chart developed using the fetched data that tell an interesting story**
- [x] **Clicking on an item in the list view displays more details about it**
- [x] **Clicking on an item has a direct, unique link to that item's detail view page**

The following **optional** features are implemented:

- [ ] The site's customized dashboard contains more content that explains what is interesting about the data
- [ ] The site allows users to toggle between different data visualizations

The following **additional** features are implemented:

- [x] Utilization of Tailwind for a better UI design
- [x] Responsive sidebar for an enhanced experience
